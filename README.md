
### Clone the repository

### Add .env file

```
SERVER_PORT=3000
DB_PASSWORD=
DB_USERNAME=

PGADMIN_DEFAULT_EMAIL=
PGADMIN_DEFAULT_PASSWORD=

AUTH0_AUDIENCE=items_endpoints

```

